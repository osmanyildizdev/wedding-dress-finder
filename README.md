# Luma Bridal

Luma Bridal is a modern and elegant bridal fashion website designed to showcase wedding dress collections through a clean, responsive, and user-friendly interface.

## Features

- Responsive design for all devices
- Modern and elegant layout
- Smooth navigation experience
- Organized and maintainable code structure
- Optimized visual presentation

## Technologies

- HTML
- CSS
- JavaScript

## Project Goal

This project was created to demonstrate modern front-end development practices while delivering a visually appealing and responsive user experience.
